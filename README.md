## 🔮 ميزات البوت
* 🔑 **ربط سريع**: توليد كود رقمي مباشر لربط رقم الهاتف تلقائياً.
* 🛡️ **نظام النقابات**: تحكم كامل برتب الجروبات، المشرفين، والأعضاء (طرد، قفل، ترحيب).
* 🎬 **ميديا متطورة**: تحميل الفيديوهات المزخرفة والملفات عبر خدمات الـ API.
* 🎨 **أوامر مزخرفة**: قائمة أوامر احترافية ومنسقة بالكامل تزيد عن 1000 أمر مستقبلي.

## 📥 طريقة التثبيت والتشغيل في Termux

انسخ الأوامر التالية بالترتيب وصوبها داخل تطبيق تيرموكس لديك:

```bash
apt update && apt upgrade -y
pkg install git nodejs -y
git clone https://github.com
cd kakashi-bot
npm install
node index.js
```
